## Key Meta Finding: Factor-Implied vs Tentative Probability

On 101 resolved binary forecasts across all versions:
- **Tentative (agent's final)**: Brier 0.2050
- **Factor-implied (sigmoid of factor sum)**: Brier 0.2086
- **Delta**: +0.0036 (factor is worse)

In every top-10 divergence case, the agent's moderation was correct. The prompt was incorrectly
warning against this pattern. Updated prompt to soften the guidance: gaps are acceptable, but
>15pp gaps should be captured as explicit factors.
